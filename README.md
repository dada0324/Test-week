# Test-Week Study Prompt

## Introduction
This project provides a **customized AI prompt** designed to maximize students' learning efficiency during Test Week. When you upload extensive lecture materials, the AI analyzes visual formatting to extract core concepts and chunks them into digestible summaries.

## Key Features
- **Format-Based Importance Analysis**: The AI recognizes bold text, italics, and font size differences in your uploaded documents to accurately identify key terms and crucial sentences.
- **Smart Chunking**: Even with massive amounts of text, the AI automatically divides the content into manageable portions to help maintain your focus.
- **Interactive Continuation (`Next` command)**: After studying the first summary, simply type `next` in the chat to seamlessly receive the summary for the following section.

## How to Use
1. Copy the prompt text provided in this repository.
2. Paste the prompt along with your study materials into an AI language model (ChatGPT, Gemini, Claude, etc.).
3. Study the first chunk of the summary, which the AI extracts based on text formatting.
4. Type `next` whenever you are ready to proceed to the next portion of the summary.
